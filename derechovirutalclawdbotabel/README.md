# derechovirutalclawdbotabel

Backup automático de automatizaciones (cron jobs) del bot de Abel — Derecho Virtual.

- **Frecuencia:** Cada 12 horas
- **Archivo principal:** `latest.json` (siempre el más reciente)
- **Histórico:** `backup-YYYY-MM-DD.json` (un archivo por día)
- **Log:** `backup-log.md`
